# bocphysics API reference

`bocphysics` is a 2D rigid-body physics engine written in Python on top of the
[`bocpy`](https://pypi.org/project/bocpy/) Behavior-Oriented Concurrency
library. It runs as an interactive simulation and doubles as a teaching aid for
learning to program with BOC.

This site is the generated **API reference**. For a getting-started guide and
the command-line options, see the
[README](https://github.com/matajoh/boc-physics#readme); the narrative tutorial
that walks from rigid-body physics through the BOC parallelisation journey lives
under `docs/tutorial/` in the repository.
